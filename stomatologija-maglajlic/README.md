# Stomatologija Maglajlić — Web

Moderna, responzivna jednostranična web prezentacija za stomatološku ordinaciju
**Stomatologija Maglajlić** iz Banje Luke. Sav sadržaj (tekstovi, usluge, slike)
preuzet je sa originalnog sajta i redizajniran u savremenom stilu sa animacijama.

## ✨ Karakteristike

- Moderan, čist dizajn sa gradientima i „glassmorphism“ headerom
- Animacije pri skrolanju, animirani brojači, plutajuće kartice i hover efekti
- Potpuno responzivan (mobitel / tablet / desktop) sa hamburger menijem
- Sekcije: Početna, O nama, Usluge (9 grana), Tim, Kontakt sa Google mapom
- Poštuje `prefers-reduced-motion`, semantički HTML, SEO/OG meta tagovi
- Bez build koraka — čisti HTML/CSS/JS (Vanilla), spremno za GitHub Pages

## 🚀 Lokalno pokretanje

```bash
# bilo koji statički server, npr.:
python3 -m http.server 8000
# pa otvorite http://localhost:8000
```
