# API

The backend exposes a versioned REST API under `/api/v1`. The authoritative reference is the OpenAPI document generated by FastAPI:

- Local: <http://localhost:8000/api/v1/openapi.json>
- Swagger UI: <http://localhost:8000/docs>
- ReDoc: <http://localhost:8000/redoc>

## Conventions

- All endpoints require a valid Supabase JWT in `Authorization: Bearer <token>` unless explicitly public.
- The active tenant is selected via the `X-Company-Id` header. The backend rejects requests where the JWT subject has no active membership for that company.
- Pagination uses `?page=` and `?page_size=` query parameters and the `Paginated<T>` envelope from `shared/types/api.ts`.
- Timestamps are ISO 8601 with timezone. Monetary fields are decimal strings to avoid float drift.
- Errors are returned as `{ "detail": string, "code"?: string }` with the appropriate HTTP status.

## Resource map

| Path | Methods | Purpose |
| --- | --- | --- |
| `/auth/me` | GET | Current user, memberships, active company. |
| `/companies` | GET, POST | List companies the user belongs to, create a new one. |
| `/companies/{id}` | GET, PATCH | Read or update a company. |
| `/companies/{id}/settings` | GET, PUT | Company settings (currency, locale, signature, footer). |
| `/customers` | GET, POST | List or create customers. |
| `/customers/{id}` | GET, PATCH, DELETE | Customer CRUD. |
| `/inquiries` | GET, POST | List or create inquiries. |
| `/inquiries/{id}` | GET, PATCH, DELETE | Inquiry CRUD. |
| `/inquiries/{id}/images` | GET, POST | List or upload inquiry images (multipart). |
| `/inquiries/{id}/analyze` | POST | Trigger AI vision analysis. |
| `/price-lists` | GET, POST | List or create price lists. |
| `/price-lists/{id}/items` | GET, POST | Items inside a list. |
| `/price-items/{id}` | PATCH, DELETE | Update or remove a price item. |
| `/quotes` | GET, POST | List or create quotes. |
| `/quotes/{id}` | GET, PATCH, DELETE | Quote CRUD. |
| `/quotes/{id}/positions` | GET, PUT | Replace positions for a quote. |
| `/quotes/{id}/pdf` | GET | Render PDF. |

For exact request and response shapes, regenerate clients from the OpenAPI document or read `shared/types/api.ts`.
